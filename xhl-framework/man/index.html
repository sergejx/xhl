Extensible Host Language

A Base for Your DSL

Extensible Host Language (XHL) is a generic syntax and a framework for building domain-specific languages. Its goal is to provide a foundation and an infrastructure for DSLs in a similar way as XML provides it for data formats and markup languages.

The syntax provides basic elements like numbers, strings, lists and maps to represent data literals in programs. It also provides shapes for your language constructs — combinations and blocks and symbols for expressing names.

The framework provides an extensible interpreter, that allow to define meaning of language elements. It also allows to express properties of language using langage schema.

Source codes

Source codes of XHL framework are available at GitHub under the Apache License Version 2.0. Note that the implementation is in early states of development and is not ready for production usage.
